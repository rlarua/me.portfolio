# 디자인 가이드 (Design Guide)

이 문서는 김명겸 개발자의 2026년 포트폴리오를 기반으로 한 디자인 시스템 명세입니다. "Full-Cycle Product Engineer"로서의 전문성과 신뢰감을 전달하기 위해 설계되었습니다.

## 1. 디자인 원칙 (Design Principles)

* **Architectural Clarity (구조적 명확성):** 불필요한 장식을 배제하고 그리드 기반의 견고한 구조를 지향합니다.
* **Minimalist & Premium (미니멀 & 프리미엄):** 넓은 여백(Whitespace)과 높은 대비를 통해 콘텐츠에 집중할 수 있는 고급스러운 룩앤필을 유지합니다.
* **Result-Oriented (결과 중심):** 특정 강조색(Sunset Gold)을 통해 성과와 수치를 직관적으로 노출합니다.

---

## 2. 컬러 팔레트 (Color Palette)

포트폴리오에서 추출된 핵심 컬러 시스템입니다.

### 2.1. Primary Colors

| Color | Hex Code | Usage |
| :--- | :--- | :--- |
| **Pure White** | `#FFFFFF` | 배경, 기본 레이아웃 |
| **Slate Gray** | `#64748B` | 보조 텍스트, 아키텍처 배경 |
| **Charcoal Black** | `#1A1A1A` | 메인 타이포그래피, 구조물 |

### 2.2. Accent Colors

| Color | Hex Code | Usage |
| :--- | :--- | :--- |
| **Sunset Gold** | `#FF9F0A` | 성과(Result/Outcome) 헤더, 강조 포인트 |
| **Tech Cyan** | `#00E5FF` | 기술 스택(Skills), 데이터 시각화 |

---

## 3. 타이포그래피 (Typography)

신뢰감 있고 현대적인 Sans-serif 서체 시스템을 사용합니다.

* **Primary Font:** `Pretendard` 또는 `Apple SD Gothic Neo` (한글), `Inter` (영문)
* **Header (Extra-Bold):** 섹션 타이틀 및 핵심 키워드
* **Body (Regular/Medium):** 설명 및 상세 내용 (가독성 중점)

---

## 4. UI 컴포넌트 가이드 (UI Component Guide)

### 4.1. 카드 (Cards)

* **Background:** `#FFFFFF` (White) 또는 매우 밝은 그레이
* **Border:** 미세한 그림자(Subtle Shadow) 또는 1px 회색 테두리
* **Style:** 모서리를 살짝 둥글게(Radius: 8px~12px) 처리하여 현대적인 느낌 부여

### 4.2. 버튼 & 링크 (Buttons & Links)

* **Primary:** `Sunset Gold` (#FF9F0A) 배경에 흰색 텍스트
* **Secondary:** `Charcoal Black` (#1A1A1A) 테두리에 검정 텍스트
* **Hover Effect:** 불투명도 조절 또는 명도 변화를 통한 부드러운 인터랙션

---

## 5. 레이아웃 & 간격 (Layout & Spacing)

* **Grid System:** 12컬럼 그리드 시스템을 권장합니다.
* **Whitespace:** 섹션 간 충분한 간격을 두어 시각적 피로도를 낮추고 정보의 위계를 명확히 합니다.
